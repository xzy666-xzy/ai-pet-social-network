# 🐶 WePet – AI Pet Social Network

WePet is an AI-powered pet social networking platform where pet owners can connect their pets with suitable friends based on personality, activity level, and interests.

The platform includes AI matching, AI pet doctor consultation, social discovery, and chat features.

---

# ✨ Features

### 🐾 AI Pet Matching
Find the best pet friends using an AI recommendation system.

- Match pets based on breed, age, personality, and activity level
- Personalized match scores
- Recommended reasons generated by AI

---

### 🧑‍⚕️ AI Pet Doctor
Consult an AI-powered pet health assistant.

- Ask pet health questions
- Get basic pet care suggestions
- Powered by OpenAI API

---

### 🌍 Explore
Discover pet-related activities and locations.

- Pet-friendly parks
- Pet cafes
- Community events
- Map-based discovery

---

### 💬 Chat
Chat with other pet owners.

- Real-time conversation interface
- User conversation list
- Messaging system

---

### 👤 Profile
Manage pet and owner information.

- Pet breed
- Pet age
- Owner profile
- Membership status

---
